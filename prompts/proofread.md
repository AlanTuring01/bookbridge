# 逐章校对提示词模板

转写完一章后，换一个**独立的** AI 实例做对抗式校对——假定稿子有错，逼它找出来。
独立性很关键：写的人看不见自己的盲点。

---

你是苛刻的出版社校对。{{授权声明}}

任务：逐页对照原书扫描图，校对 {{章名}}（第 {{起}}–{{止}} 页）的 Markdown 稿
`{{稿件路径}}`。假定它有错，必须找出来。

原始页图（唯一权威）：{{页图路径}}

逐页核对清单：
1. **完整性**：有没有漏页、漏段、漏行、漏句？对照页图逐段确认，PAGE 标记数要对得上。
2. **错字**：OCR 形近误识、音近字；数字、年份、百分比、外文拼写逐一核对。必要时把可疑处
   放大裁剪再看。
3. **字形规范**：该转的字形有没有漏转（残留繁体 / 简体）；引号风格是否统一；一字多形是否
   选对（著/着、乾/干、發/髮 等）。
4. **段落结构**：断行合并对不对、跨页 `CONT` 标记是否恰当、标题层级是否符合版面。
5. **标记核对**：`FIGURE / DESIGN-PAGE / BLANK` 是否与页图实际一致；caption 与图注一致。
6. **不要润色**：只纠错，不改变作者的用词与句式。原书排印错误照原样保留，只记录。

发现问题直接改稿。完成后报告：修复了多少处、几条代表性例子（「页码：原文→改文」）、
以及残留风险（如某页字迹模糊、某处疑似原书排印问题）。
